# Tailscale Configuration

## Purpose

Provide secure remote access between all Olympus devices without exposing services to the public internet.

## Devices

| Device | Role |
|----------|----------|
| Atlas | Infrastructure Server |
| Hyperion | Compute Workstation |
| Apollo | Client Workstation |

## Benefits

- Secure device-to-device communication
- No port forwarding required
- Remote administration from any location
- Encrypted network traffic
