# Docker

## Overview

Docker was installed on Atlas to provide a consistent and isolated environment for running infrastructure services.

## Purpose

Docker allows applications to be deployed as containers without directly modifying the host operating system.

Benefits include:

- Simplified deployments
- Service isolation
- Easier maintenance
- Reproducible environments
- Simplified backups and migrations
