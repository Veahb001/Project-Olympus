# Hyperion

## Purpose

High performance workstation used for compute intensive workloads.

## Wake-on-LAN Configuration

### BIOS

Enabled:

- Wake on LAN
- Resume by PCI-E
- Power On by LAN

### Windows

Network Adapter:

- Wake on Magic Packet
- Allow device to wake computer
- Only allow magic packet

### Power Configuration

Disabled:

- Fast Startup

## Outcome

Hyperion can be remotely powered on using Wake-on-LAN.
